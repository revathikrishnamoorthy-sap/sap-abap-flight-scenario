# SAP ABAP Weiterbildung – Praxisprojekt (Flight Scenario)

## Projektübersicht
Dieses Projekt wurde im Rahmen meiner SAP-Weiterbildung umgesetzt.  
Ziel war es, praktische Erfahrung in der Backend-Entwicklung mit ABAP OO sowie in der Datenmodellierung mit CDS Views zu sammeln.

## Inhalte / Umsetzung
- Erstellung eigener Datenbanktabellen (Carrier, Connection, Flight)
- Entwicklung von ABAP OO Klassen zur Befüllung der Tabellen mit Beispieldaten
- Verwendung von Open SQL inkl. JOINs zur Auswertung zusammengehöriger Daten
- Erstellung von CDS Views mit Assoziationen und Prüfung der Ergebnisse per Data Preview

## Technologien
- ABAP OO  
- Open SQL (JOINs)  
- CDS Views & Assoziationen  
- Eclipse ADT (ABAP Development Tools)  
- SAP S/4HANA (Trainingssystem)

## Ergebnis
Das Projekt zeigt die Umsetzung eines einfachen SAP-Backend-Szenarios mit funktionierender Datenmodellierung, Datenbefüllung und auswertbaren Abfragen. Die Projektdokumentation (PDF) enthält Screenshots der Implementierung.
